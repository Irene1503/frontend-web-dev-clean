13 Types of CSS Selectors

# Selector Type Example Description
1 Universal Selector * { margin: 0; } Selects all elements on the page.
2 Type Selector p { font-size: 16px; } Selects elements by tag name.
3 Class Selector .card { border: 1px solid gray; } Selects all elements with a specific class.
4 ID Selector #header { background: navy; } Selects a single element by ID.
5 Grouping Selector h1, h2, h3 { font-family: Arial; } Applies styles to multiple selectors.
6 Descendant Selector nav ul li { list-style: none; } Targets elements nested inside another.
7 Child Selector div > p { color: blue; } Targets direct children only.
8 Adjacent Sibling Selector h1 + p { font-style: italic; } Targets the next sibling only.
9 General Sibling Selector h1 ~ p { color: green; } Selects all siblings after a specific element.
10 Attribute Selector input[type="text"] { border: 1px solid black; } Targets elements with a specific attribute value.
11 Pseudo-class Selector a:hover { text-decoration: underline; } Targets elements in a specific state (hover, focus, etc).
12 Pseudo-element Selector p::first-letter { font-size: 2em; } Styles part of an element's content.
13 :not Selector input:not([type="submit"]) { background: lightyellow; } Excludes certain elements from being styled.
